Função CalcularHeuristica(estado)

    N ← tamanho do estado

    criar vetor conflito[N]

    para i ← 0 até N-1 faça
        conflito[i] ← falso

    para i ← 0 até N-1 faça

        para j ← i+1 até N-1 faça

            se estado[i] = estado[j] então

                conflito[i] ← verdadeiro
                conflito[j] ← verdadeiro

            se abs(i-j) = abs(estado[i]-estado[j]) então

                conflito[i] ← verdadeiro
                conflito[j] ← verdadeiro

    h ← 0

    para i ← 0 até N-1 faça

        se conflito[i] = verdadeiro então
            h ← h + 1

    retornar h
